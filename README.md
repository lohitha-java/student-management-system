# student-management-system
A simple Student Management System  built using Core java and SQL.
This project performs basic CRUD operations to manage student records.
## Features
- Add new student
- Update student details
- Delete student
- View all students
## Technologies Used
- Core Java
- SQL
- JDBC
## Project Description
This is application allows users to manage student data Such as ID, Name, Email, and Course using a console-based interface
